  • Display: flex — enables flexbox layout on the container.

  • Flex-direction — sets the direction of items (row, column, etc.).

  • Flex-wrap — allows items to wrap onto multiple lines.

  • Justify-content — aligns items horizontally (space-around in this example).

  • Align-items — aligns items vertically (center in this example).

  • Gap — sets spacing between items.

  • Flex — sets grow, shrink, and basis for items.

  • Flex-grow — makes an item grow more than others.

  • Align-self — overrides vertical alignment for a single item.
